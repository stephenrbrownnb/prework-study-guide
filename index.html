HTML

  • The head element contains information for the browserabout the webpage.
  • Most of the information in Head isn't visible to the user.
  • However the "Title" in header is visible in the browser tab.
  • The body element represents the visible content shown to the user.

CSS

  • CSS can be used to make rules for styling a webpage instead of adding code to each individual line
  • A margin indicates how much space we want around hte outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A is used to upload files to your branch. the add is the command and the -A represents all files.
  • git commit (-m) updates files to github the -m allows a message to be associated with the update.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript reads code from Top to bottom.
